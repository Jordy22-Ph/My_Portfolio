프로젝트 개요

고혈압, 당뇨, 만성신장질환 3가지 만성질환 환자를 대상으로, AI 위험도 예측과 LLM 기반 건강 조언, 챌린지/펫 보상 시스템을 결합한 생활습관 관리 웹 서비스입니다.

사용자가 혈압·혈당 등 건강 수치를 입력하면 AI가 질환 위험도를 예측(최대 AUC 0.957)하고, OpenAI API 기반 LLM이 개인화된 건강 조언을 제공합니다. 챌린지를 달성하면 포인트와 펫 성장 보상을 받아 지속적인 건강 관리를 유도합니다.

부트캠프 최종 프로젝트로 all4health.kro.kr에 실서비스 배포가 완료되었습니다.

AI 맞춤 건강 조언
OpenAI API 기반. 예측된 위험도·주요 위험 인자·건강 수치를 컨텍스트로 주입해 오늘의 맞춤 생활습관 개선 조언을 자동 생성합니다.

챌린지 & 펫 보상 시스템
일일/주간 건강 챌린지 달성 시 포인트 획득, 펫 성장 보상으로 장기적 건강 관리 동기 부여.

건강 수치 트래킹

인증 & 보안

나의 역할

프론트엔드 개발 — React + TypeScript로 주요 페이지 UI 구현 (건강 수치 입력, 결과 대시보드, 챌린지 화면)
AI 모델 전처리 파이프라인 담당 — 원본 의료 데이터 정제, 피처 선택, 모델 입력 형식 정규화
XGBoost + LightGBM 앙상블 모델 학습 및 API 연동
FastAPI 엔드포인트 일부 구현 — 건강 수치 저장, 예측 결과 반환
Docker Compose 설정 및 AWS EC2 배포 참여
