프로젝트 개요

영국 HFEA(Human Fertilisation and Embryology Authority)의 난임 시술 데이터 약 26만 건을 활용하여 임신 성공 여부를 예측하는 이진 분류 모델을 개발한 Dacon 경진대회 프로젝트입니다.

ML(XGBoost, LightGBM) 1단계와 DL(TabNet) 2단계를 결합한 2단계 스태킹 앙상블 전략을 사용했으며, 팀 내에서 Data Leakage를 최초로 발견·수정하고 TabNet 최적화를 담당했습니다.

핵심 기여 사항

Data Leakage 발견 및 수정
EDA 과정에서 훈련 데이터에 타깃과 직결된 파생 변수가 포함되어 있음을 팀 내 최초로 발견했습니다. 해당 변수를 제거하고 파이프라인을 재구성하여 과적합을 제거하고 실제 일반화 성능을 보장했습니다.
TabNet 범주형 임베딩 최적화 (기여도 0.4% → 10.3%)
초기 TabNet 모델은 범주형 변수를 기본 설정으로 처리해 앙상블 기여도가 0.4%에 불과했습니다. 범주형 변수의 카디널리티(cardinality)에 따라 임베딩 차원을 세밀하게 조정하고, cat_dims/cat_emb_dim 파라미터를 Optuna로 최적화하여 기여도를 10.3%까지 끌어올렸습니다.

모델 파이프라인

01
데이터 전처리 및 피처 엔지니어링
결측치 처리, 레이블 인코딩, Data Leakage 변수 제거, 도메인 지식 기반 파생 변수 생성
02
1단계: ML 앙상블 (OOF)
XGBoost + LightGBM을 K-Fold OOF 방식으로 학습. 각 Fold의 예측값을 2단계 입력 피처로 사용.
03
2단계: TabNet (DL 메타 학습)
1단계 OOF 예측 + 원본 피처를 입력으로 TabNet 메타 모델 학습. 범주형 임베딩 최적화로 기여도 대폭 향상.
04
앙상블 가중치 최적화
ML 앙상블 vs. TabNet 출력의 가중 평균 비율을 Optuna로 최적화하여 최종 예측 생성.
