프로젝트 개요

국가건강검진 공공데이터를 활용하여 흡연자와 비흡연자의 건강 지표 차이를 탐색적 데이터 분석(EDA)과 통계 검정을 통해 비교 분석한 프로젝트입니다.

단순 시각화에 그치지 않고 카이제곱 검정, t-test, Mann-Whitney U test 등 적절한 통계 기법을 선택하여 4개의 가설을 수립하고 검정했습니다. 4개 중 3개의 가설에서 통계적으로 유의미한 차이를 확인했습니다.

검증한 가설

가설 01
혈압과 흡연 여부의 관계
흡연자는 비흡연자에 비해 수축기 혈압과 이완기 혈압이 유의미하게 높을 것이다.
→ t-test 적용. 수축기 혈압에서 유의미한 차이 확인.
✓ 채택
가설 02
간 기능 지표(AST/ALT)와 흡연
흡연자는 비흡연자보다 간 기능 지표(AST, ALT)가 높을 것이다.
→ Mann-Whitney U test 적용. 유의미한 차이 확인.
✓ 채택
가설 03
BMI와 흡연 여부의 관계
흡연자와 비흡연자 간 BMI 분포에 유의미한 차이가 있을 것이다.
→ t-test 적용. 유의미한 차이 확인.
✓ 채택
가설 04
청력 이상과 흡연 여부
흡연자는 비흡연자보다 청력 이상(이상 소견 비율) 비율이 높을 것이다.
→ 카이제곱 검정 적용. 유의미한 차이 확인되지 않음.
✗ 기각

나의 역할

데이터 전처리 전담 — 결측치 처리, 이상치 탐지, 범주형/연속형 변수 분리
가설 수립 및 통계 기법 선택 — 각 가설의 데이터 특성에 맞는 검정 방법 결정
시각화 코드 작성 — Matplotlib/Seaborn으로 분포 비교, 박스플롯, 히트맵 구현
결과 해석 및 발표 자료 정리

시각화 결과물

이미지 준비 중

흡연자 vs 비흡연자 건강 지표 분포 비교 (박스플롯)

이미지 준비 중

상관관계 히트맵 — 흡연과 각 건강 지표 간 관계

사용 기술 및 방법론

데이터 분석 파이프라인
Pandas로 원본 데이터 로드 → 결측치/이상치 처리 → 흡연 여부 기준 그룹 분리 → 기술 통계 산출 → 통계 검정 (scipy.stats) → Seaborn/Matplotlib 시각화
통계 기법 선택 기준
정규성 검정(Shapiro-Wilk) 결과에 따라 모수적(t-test) vs. 비모수적(Mann-Whitney U) 방법 결정. 범주형 변수 비교는 카이제곱 검정 적용.
